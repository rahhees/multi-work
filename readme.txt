# 🌐 Personal Portfolio Website (Advanced Version)

This is a fully responsive **Personal Portfolio Website** built using **HTML, CSS, JavaScript, and Bootstrap**. It features **multiple pages** with smooth navigation and a clean, modern UI for showcasing your skills, projects, resume, and contact information.

---

## 🔥 Features

- 🧭 **Multi-page Navigation** (`Home`, `About`, `Skills`, `Projects`, `Contact`)
- 📱 **Responsive Design** (Mobile-first using Bootstrap 5)
- 🎨 **Custom Styling** with CSS for unique UI feel
- ⚡ **Interactive Elements** using JavaScript
- 🔗 **External Links** (Resume download, social profiles)
- 📍 **Embedded Google Maps** for contact/location
- 💼 **Professional layout** suitable for real-world portfolio use

## 🚀 Technologies Used

- **HTML5** – Semantic structure
- **CSS3** – Custom styling
- **JavaScript (Vanilla)** – Form validation, interactivity
- **Bootstrap 5** – Responsive grid, layout, components
- **Font Awesome** – Icons for UI enhancements

---

## 🖥️ Pages Overview

- `index.html` – Landing page with introduction & call-to-action
- `about.html` – Bio, skills, and timeline/experience
- `projects.html` – Showcasing completed projects with images & descriptions
- `contact.html` – Contact form with embedded Google Map
